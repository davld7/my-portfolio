Anime List App

Este proyecto de Android utiliza una API propia para acceder a los datos de animes almacenados en un servidor remoto con MongoDB Atlas. La API devuelve una lista de objetos JSON con la información de cada anime (Minimun SDK: API 23 - Android 6.0 Marshmallow).

Anime List API

Este proyecto utiliza el framework FastAPI para crear una API en Python y acceder a los detalles de animes de una base de datos remota en MongoDB Atlas.
